# ned: simple text editor written in nim
based on kilo: https://viewsourcecode.org/snaptoken/kilo

## Key mapping
|Key|Description|
|---|-----------|
|Ctrl-H | Move left |
|Ctrl-J | Move down |
|Ctrl-K | Move up |
|Ctrl-L | Move right |
|Ctrl-F | Move to next page |
|Ctrl-B | Move to previous page |
|Ctrl-W | Save file |
|Ctrl-S | Search word |
|Ctrl-Q | Terminate editor |
